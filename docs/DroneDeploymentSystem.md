Drone Deployment System
-----------------------

### Deployment

Deployed during movement phase. Each system deploys one per turn. Placed anywhere along ship's movement course (straight line from start to end).

Mass 3 and each system carries three drones.

Active the turn AFTER deployment.

Completely stationary.

### Recovery

### In-flight Recovery System

Must activate recovery system.

Must move within 3&Prime; of drone during course of movement. Roll to successfully recover. Cannot use any offensive weapon systems in the same turn.

| On a roll of...
| 1-2 | Unsuccessful
| 3-6 | Drone recovered
